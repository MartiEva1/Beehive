Access to a Group Chat Overview The main goal of Beehive is to provide an intuitive and easy-to-use application to join and create events. The idea behind the application is to help in meeting new people and live new experiences, it gives the possibility to chat with the other users participating to an event and to keep track of the events that you created or booked. Get Started See the tutorial Your Profile
In the Profile page the main information of the user are shown: username, date, profile picture.
  • Edit Profile Information: Yellow Edit button let you change your name, date of birth and profile picture.
  • Delete your account: Tap the yellow Edit Button and then the Delete your account option is shown in red.
  • Edit Password: shown in the main page of the profile.
  • Log-out: shown in the main page of the profile.
 Participate to an event
In the home page you can select the preferred category.
Then, a card of an event is shown, for each card you can:
  • Accept the event: swiping right or clicking the green check button.
  • Postpone the decision: swiping up or clicking the yellow clock button.
  • Reject the event: swiping left or clicking the red X button.
 Add an event To add an event you have to click the yellow + button in the home page, compile the presented form and press the publish button.
Your event will be now in the Event page in “created” session. Leave or Delete an event
If you created an event that you want to Delete:
  • Go to Event page (from the tap bar)
  • Click on “created” session
  • Click on Delete button of the chosen event
If you booked an event that you want to Leave:
  • Go to Event page (from the tap bar)
  • Click on "booked" session
  • Click on Leave button of the chosen event
 Edit an event
If you created an event that you want to Edit:
  • Go to Event page (from the tap bar)
  • Click on “created” session
  • Click on Edit button of the chosen event
  • Edit the information
  • Click on Save button
 Access to a Group Chat In the home page press the chat bubbles up right in the page, you will have a list of chats with one chat for each event you booked, you can click on the one you prefer and chat with the participants.
